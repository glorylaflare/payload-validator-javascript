# ✅ Validador de Payload em JavaScript

## Descrição
Este projeto fornece um conjunto de ferramentas para validar payloads JSON em aplicações JavaScript, facilitando a verificação de dados de entrada de forma eficiente e reutilizável.


## Pré-requisitos
Antes de começar, certifique-se de que você tem instalado:
- Node.js
- npm
- Git
- VS Code

## Como Iniciar
Siga estes passos para configurar e executar o projeto localmente:

1. **Clone o repositório:**
   ```
   git clone https://github.com/glorylaflare/payload-validator-js
   cd payload-validator-js
   ```

2. **Instale as dependências:**
   ```
   npm install
   ```

3. **Execute o projeto:**
   ```
   npm start
   ```

## Estrutura do Projeto
